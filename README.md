# WSP-ARCHIVE 

Designing Frontend for [Whatsapp Chat Exporter](https://github.com/KnugiHK/WhatsApp-Chat-Exporter)


## Project Status

This project is currently on hold.

While working on the WhatsApp data extraction and JSON conversion, I ran into multiple issues related to encrypted databases (Crypt15) and extremely inconsistent JSON structures generated from the extracted data. Properly solving this would require deeper work in cryptography, database reverse engineering, and building a reliable parser for messy and inconsistent data formats.

At this point, I do not have the time to go deep into the cryptography and reverse engineering required to continue this project properly. Rather than leaving the repository unfinished without explanation, I am marking the project as paused.

I may return to this project in the future if I explore more cryptography, encryption formats, or WhatsApp database structures. If I do, development will resume here.

For now, consider this project experimental and incomplete.
